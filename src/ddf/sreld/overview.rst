Software release overview
*************************

The released software is defined by the |SCF| ${/pkg/deployment/doc-package-manual:/cite}.  The
${/glossary/qdp:/term} package version is ${.:/component/package-version}.  The
QDP was built for the
``${.:/component/arch}/${.:/component/bsp}${.:/component/config:slash}``
variant.

For users of the QDP and ${/glossary/rtems:/term} it is recommended to get
started by reading the Software Configuration File ${/pkg/deployment/doc-package-manual:/cite}.  For using
RTEMS in general, please refer to the RTEMS User Manual ${/ref/rtems/user:/cite}
and the RTEMS Classic API Guide ${/ref/rtems/c-user:/cite}.
