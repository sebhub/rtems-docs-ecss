Annex D - Source code changes from package version 5 to 6
#########################################################

${.:/push-enabled-by:compare-sources}
This annex to the ${/glossary/svr:/term} contains the changes in the source
code of the pre-qualified features set from package version 5 to 6.  The
pre-qualified feature set consists of all header and source files necessary to
build and use :file:`start.o`, :file:`librtemsbsp.a`, and
:file:`librtemscpu.a`.  In particular, test files are excluded from the
comparison.  The changes between the versions are presented in categories.
Each substantial change is reviewed.
